Function Declaration

This is where you declare how your function works. Think of it as a blueprint. Why a blueprint? Usually, we start with a problem we want to solve. Our function declaration is the blueprint to how we want to solve the problem in the future.
function
addNumbers (num1, num2)
{
const sum = num1 + num2; console.log(sum); // prints sum to the console
}

Function Call

This is how you use your function. The function call consists of the function name and the braces containing the values you want to pass to the function. You can use your function anywhere in the same scope (e.g. file) you declared it in, even above its initial declaration.
addNumbers (2, 3)
// 5 will be printed to the console
